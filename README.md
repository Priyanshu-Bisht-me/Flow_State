## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software required - pure HTML, CSS, and JavaScript!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/flowstate.git
   cd flowstate
   ```

2. **Open the application**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # or
   start index.html
   ```

3. **Start using FlowState!**
   - Navigate to the dashboard by clicking "Get Started"
   - Begin tracking your time and building habits

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables
- **Storage**: LocalStorage for data persistence
- **Design**: Neumorphism, Glassmorphism, Responsive Design
- **Icons**: Emoji icons for lightweight, universal compatibility

## 🎨 Design System

### Color Palette
- **Primary**: `#667eea` (Soft Blue)
- **Secondary**: `#764ba2` (Purple)
- **Accent**: `#f093fb` (Pink)
- **Success**: `#48bb78` (Green)
- **Warning**: `#ed8936` (Orange)
- **Error**: `#f56565` (Red)

### Typography
- **Font Family**: Poppins (Google Fonts)
- **Weights**: 300, 400, 500, 600, 700
- **Responsive**: Clamp-based sizing for perfect scaling

### Effects
- **Neumorphism**: Soft, inset/outset shadows
- **Glassmorphism**: Backdrop blur with transparency
- **Animations**: Smooth cubic-bezier transitions

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px
